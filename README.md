# Dropbox Fetcher

## Getting Started

### Prerequisites

+ [python3](https://www.python.org/download/releases/3.0/)
+ [virtualenv](https://virtualenv.pypa.io/en/stable/)
+ [fabric](http://www.fabfile.org/)

### Configure

```bash
cp config.yml{.template}
# edit config.yml 
```

### Start

```bash
fab fetch
```
